# IMR (Iterative Matrix Reduction)

IMR is a computational tool developed for genomic surveillance of SARS-CoV-2 in wastewater and environmental water samples.

The algorithm estimates the relative abundance of SARS-CoV-2 variants from single nucleotide variant (SNV) frequency data obtained from heterogeneous sequence data containing multiple co-circulating variants. This approach enables lineage composition analysis from mixed-population sequencing data, including datasets with variable sequencing depth and genome coverage.

IMR uses lineage constellation definitions provided by [Alcov](https://github.com/Ellmen/alcov) for SARS-CoV-2 variant identification and abundance estimation.

## Applications

* Wastewater-based epidemiology (WBE)
* Environmental genomic surveillance
* SARS-CoV-2 variant tracking
* Analysis of mixed-population sequencing data
